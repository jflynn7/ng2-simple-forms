Grouped Radio Buttons

Standard radio buttons, but with option groupings

A grouped radio input is exactly the same as standard radio inputs, except they also offer groupings for the options. That is to say, the options are still part of the same input, but in a similar way to how grouped dropdowns work, they also have group headings for display.
