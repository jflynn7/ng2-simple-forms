Label

Dynamic classes added to labels

Sometimes, you may wish to apply css to labels depending on the form input state. SimpleForms adds certain classes based on the state of the input to make this easier, simply override the styles you want.

NB. By default, SimpleForms adds NO styling to these classes, so if you don't override them, there'll be no visible change) 

.input-focussed

Applied when the input field corresponding to the label has focus.

.has-value

Applied when the input field corresponding to the label has a value that isn't null or undefined

.valid

Applied when the input field corresponding to the label is valid and has been touched.

.invalid

Applied when the input field corresponding to the label is invalid and has been touched.
