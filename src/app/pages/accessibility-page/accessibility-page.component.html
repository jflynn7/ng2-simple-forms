ARIA labels that are consistently supported across all components are applied using a custom @Directive which gets passed the components elementData and formGroup to do its thing. If you dive into the code, you'll see it applied to each component using: 

[appDefaultAccessibility]="getElementData()"

getElementData() simply returns an object of type: {{'{'}} elementData: FormElement, formGroup: FormGroup {{'}'}}
